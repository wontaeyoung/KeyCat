# 한 줄 소개

키보드 상품 특화 커머스 플랫폼 앱

<br><br>

# 핵심 기능

- 로그인 / 회원가입 / 판매자 사업 권한 인증
- 상품 판매글 작성 / 조회 / 삭제 기능
- 상품 결제 기능
- 상품 리뷰 작성 / 조회 / 삭제 기능
- 북마크 / 장바구니
- 프로필 수정 기능
- 팔로우 기능

<br><br>
